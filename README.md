# Course-evaluation-script
The script automatically selects answers for the course evaluation form so you don't have to click through every question manually. It simply fills the form with predefined choices to speed up the process.

# How to use:
1. Open the course evaluation page.
2. Press F12 to open the browser Developer Tools.
3. Go to the Console tab.
4. Copy the one-liner from script.txt.
5. Paste it into the console and press Enter.
6. The script will automatically complete the evaluation fields.
